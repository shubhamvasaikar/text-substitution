Demostrar algunas características HTML

HTML es realmente un lenguaje muy sencillo. Se compone de texto ordinario, con los comandos que están encerrados por parte de los personajes, o entre un "&" y un ";".

Usted realmente no necesita saber mucho de HTML para crear una página, porque se puede copiar trozos de HTML procedentes de otras páginas web que haga lo que quiera, a continuación, cambiar el texto!

Esta página muestra a la izquierda, tal como aparece en su navegador, y el correspondiente código HTML que aparece a la derecha. Los comandos HTML están vinculados a las explicaciones de lo que hacen.

Los Saltos De Línea

HTML normalmente no utilizar saltos de línea para texto normal. Un espacio en blanco de cualquier tamaño es tratada como un único espacio. Esto es porque el autor de la página no tiene forma de saber el tamaño de la pantalla del lector, o qué tipo de tamaño que tendrá el navegador configurado para.
